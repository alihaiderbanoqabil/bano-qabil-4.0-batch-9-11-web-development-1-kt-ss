CSS min-width Explained

What is min-width?

The min-width property in CSS sets the minimum width of an element. It prevents the element from becoming narrower than the specified value, even when the parent container is resized.

Example

Resize the window to see how this box respects min-width! 
.example-box {
  min-width: 300px;
  background-color: #4caf50;
  color: white;
  padding: 20px;
  text-align: center;
}
